## Levantamento das Fontes de Dados

### Fonte principal

1. INPE Queimadas

    Possíveis datasets:
    - focos ativos
    - risco de fogo
    - áreas queimadas
    - histórico de focos


### Fontes Complementares

1. IBGE

    Possíveis datasets:
    - estados
    - municípios
    - malhas geográficas

2. MapBiomas

    Possíveis datasets:
    - uso e cobertura do solo

3. INMET

    Possíveis datasets:
    - chuva
    - temperatura
    - umidade
